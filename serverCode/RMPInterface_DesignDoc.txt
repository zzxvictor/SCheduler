update rate my professor score for all instructors.
functionalities:

getInstructors: get the list of instructors for the given semester
output format: [(prof name, course prefix, coresponding department name),(...),()]
the purpose of storing course prefix and department name is to distinguish professors with the same names.

crawleRMP: search RMP using keywords generated by getInstructors. For instance, search RMP with USC+Computer Science+Aaron Cote.
The result is save into a json file with the following format:
{
	profName1: {deparmentName1:score1, departmentName2: score2,} // if multiple professors are found 
	profName2: {departmentName1: score1)
}
The score is generated by combining avg. rating with the number of raters.
